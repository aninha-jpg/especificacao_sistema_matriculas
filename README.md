# 📚 Especificação de Sistema de Matrículas

Repositório dedicado à documentação e especificação de requisitos para um sistema de gestão de matrículas acadêmicas. Este projeto foca-se na análise de sistemas, levantamento de requisitos e modelagem de processos.

## 📋 Sobre o Projeto

O objetivo deste sistema é automatizar o processo de inscrição de alunos em disciplinas, melhorando o desempenho em épocas de vestibulares, gerir a oferta de turmas e validar pré-requisitos, garantindo que as regras de negócio da instituição sejam respeitadas.

---

## 🛠️ Tecnologias e Ferramentas

* **Modelagem:** Excel, Draw.io e Figma.
* **Ambiente:** Ubuntu Linux

---

## 📑 Funcionalidades Principais (Escopo)

- **Painel de Candidatos e Estudantes**
- **Informações de Disciplinas**
- **Gerenciamento de Matrículas** 
- **Painel Administrativo** 
- **Painel de Pagamento**
